# Mi Curriculum (sitio estático)

Pequeña página estática para mostrar historial y currículum usando HTML, CSS y JS separados.

Cómo ver localmente:

1. Abrir `index.html` directamente en el navegador.
2. O ejecutar un servidor local (recomendado):

```bash
# usando Python 3 desde la carpeta del proyecto
python -m http.server 8000
# luego abrir http://localhost:8000
```

Archivos principales:
- index.html
- styles/style.css
- scripts/main.js
